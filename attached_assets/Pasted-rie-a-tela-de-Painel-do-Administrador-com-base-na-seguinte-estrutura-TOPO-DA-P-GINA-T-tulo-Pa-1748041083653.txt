rie a tela de Painel do Administrador com base na seguinte estrutura:

🟩 TOPO DA PÁGINA
Título: "Painel do Administrador"

🟨 RESUMO GERAL (como opção no menu lateral)
Crie uma opção no menu lateral chamada Resumo Geral

Dentro dessa página, exiba cards de KPIs com ícones e estilo visual de destaque:

Ícone	Informação	Exemplo
👤	Usuários ativos	128
🏷️	Marcas protegidas	52
📦	Casos ativos	1.247
💰	Valor potencial de indenização	R$ 1.283.000,00
🔁	Notificações emitidas no mês	402

Cada card deve puxar dados reais do sistema e ser atualizado dinamicamente.

📋 MENU LATERAL DO ADMINISTRADOR
Exiba somente para usuários com perfil “admin”

Adicione os seguintes itens no menu lateral, com submenus direcionando para páginas específicas:

📁 Usuários
Página com:

Lista de todos os usuários

Botão: “Criar novo usuário”

Filtros: por setor e por status (ativo/inativo)

Ações em cada linha: editar, redefinir senha, desativar

🔐 Permissões
Página com:

Tabela com todos os tipos de acesso

Lista de permissões por perfil (estrutura colapsável)

Botão para criar novo perfil personalizado

🕵️‍♂️ Auditoria e Histórico
Página com:

Tabela com logs de atividade

Colunas: Usuário, Ação, Data/hora

Filtros por tipo de ação, usuário, intervalo de datas

🏷️ Marcas e Clientes
Página com:

Lista de marcas cadastradas

Botão: “Adicionar nova marca”

Ações: editar vínculo com cliente, visualizar casos da marca

📄 Templates e Documentos
Página com:

Lista de modelos de documentos (como notificações)

Ações: editar texto, subir novo modelo, visualizar versão atual

⚙️ Configurações Gerais
Página com:

Campos editáveis para:

Prazo padrão de movimentação entre setores

E-mails automáticos de notificação

Chaves de integração/API (Ex: Correios, plataformas de denúncia)

🟦 DESIGN VISUAL
Seguir o mesmo estilo visual do sistema geral

Cards com:

Bordas arredondadas

Cores suaves e intuitivas (verde para positivos, vermelho para alertas)

Barra lateral colapsável

Ícones representando cada item do menu

Destaque visual para a página atual
