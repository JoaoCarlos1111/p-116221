1. 🔐 TELA DE LOGIN – CLIENTE
Rota: /login/cliente

Campos:

E-mail (type="email")

Senha (type="password")

Links adicionais:

“Esqueci minha senha” → redireciona para /recuperar-senha/cliente

Comportamento:

Autentica via endpoint exclusivo de clientes

Em caso de erro (usuário inválido ou senha incorreta), exibir mensagem clara no formulário

Após login bem-sucedido, redirecionar o cliente à dashboard vinculada à marca dele

Layout:

Seguir identidade visual do tema atual

Centralizado, com o logo no topo

Responsivo para desktop e mobile

2. 🔐 TELA DE LOGIN – TIME INTERNO (Analistas e Admin)
Rota: /login/internal

Campos:

E-mail (type="email")

Senha (type="password")

Links adicionais:

“Esqueci minha senha” → redireciona para /recuperar-senha/internal

Comportamento:

Autentica via endpoint exclusivo de usuários internos

Após autenticação, identificar o papel do usuário:

Analista de Prospecção

Analista de Verificação

Analista de Auditoria

Analista de Logística

Analista de IP Tools

Analista de Atendimento

Analista Financeiro

Administrador

Redirecionar automaticamente à dashboard adequada ao setor do analista

Em caso de erro, mostrar mensagem adequada no formulário

3. 🔁 RECUPERAÇÃO DE SENHA
Rotas:

/recuperar-senha/cliente

/recuperar-senha/internal

Campos:

E-mail (com validação básica)

Comportamento:

Ao submeter, o sistema envia um e-mail com token único e link de redefinição

O link abre uma tela de “Definir Nova Senha”, com os campos:

Nova senha

Confirmar senha

Após redefinir, redirecionar para a tela de login correspondente
